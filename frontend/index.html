About

About This Project

This AI Medical Assistant is a final-year college project exploring the intersection of artificial intelligence and healthcare accessibility. The goal is to demonstrate how conversational AI can help people access basic health information quickly and simply.

The chatbot is built using a large language model (LLM) with a custom medical knowledge retrieval system (RAG), wrapped in a clean, responsive web interface.
